Приборы учёта (ПИУ)

Внимание! Показания ИПУ необходимо передавать до 23 числа каждого месяца, независимо от того, когда вы платите!
1

Снимите показания

Записывайте только целую часть расхода (чёрные цифры на циферблате), красные цифры (литры) учитывать не нужно.

2

Выберите способ

через операционистов Сбербанка при оплате;

через систему Сбербанк-он-лайн;

3

Работа с должниками

В случае неоплаты коммунальных услуг по показаниям ИПУ или несдачи показаний ИПУ ЕЖЕМЕСЯЧНО, первый месяц начисления по коммунальных услугам будут произведены по вашему среднемесячному потреблению, а затем по утвержденным нормативам.
